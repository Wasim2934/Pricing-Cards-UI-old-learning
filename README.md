# Pricing Cards UI

> ⚠️ **Archived Notice**  
> This project was built during my initial learning phase.  
> This repository is archived and kept only for history and progress tracking.  
> A better and more professional version of this project will be built in a new repository.

A clean and modern pricing cards section built using **HTML & CSS**.  
This project displays three pricing plans — **Basic, Standard, and Premium**, with the middle card highlighted to attract user attention.

## Live Demo - https://wasim2934.github.io/Pricing-Cards-UI/

## Features

- Minimal & modern UI
- Fully responsive layout
- Hover lift animation
- Google Fonts (**Poppins**)
- Clean shadows & spacing
- Highlighted featured pricing plan

## Tech Stack

- HTML5
- CSS3
- Google Fonts
